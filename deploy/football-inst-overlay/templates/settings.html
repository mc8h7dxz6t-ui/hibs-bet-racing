Fixtures

How many days of fixtures to load on the dashboard. Pick 7 days during European finals week so late kick-offs appear. Saved in this browser and synced via ?days= on refresh. Server default if unset: {{ fetch_days }} days.

Data Quality

Minimum data % for the “Full data” filter (form, xG, odds, etc.). Higher = fewer fixtures, richer stats. Does not change how data is fetched.
